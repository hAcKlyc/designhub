Claude Design System

Build beautiful products faster

A literary salon reimagined as a product page. Warm, unhurried, and quietly intellectual. Every gray has a story.

Features

Warm Parchment Canvas

#f5f4ed background evokes premium paper. Every neutral carries yellow-brown undertones — no cold grays anywhere.

Serif for Authority

Anthropic Serif at weight 500 for all headlines. Single weight, book-title gravitas. Sans handles UI with quiet efficiency.

Ring Shadow System

0px 0px 0px 1px ring shadows create border-like depth. Whisper-soft elevation that's felt rather than seen.
