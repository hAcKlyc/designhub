Spotify Design System

Build beautiful products faster

Dark, immersive, content-first. The UI recedes into shadow so that music and art can glow.

Features

Pill & Circle Geometry

500-9999px pill buttons, 50% circles for play controls. Rounded, touch-optimized, device-like.

Content-First Dark

Near-black cocoon where album art provides the color. The UI itself is achromatic by design.

Uppercase System

Button labels in uppercase + 1.4-2px letter-spacing. A systematic label voice distinct from content.
